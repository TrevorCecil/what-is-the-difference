What's the Difference?

HTML

Hypertext Makeup Language

Create the structure

  • Controls the layout of the content
  • Provides structure of the web design
  • The fundamental building block of any page

CSS

Cascading Style Sheet

Stylize the website

  • Applies style to the webpage elements
  • Targets various screen sizes to make the webpage responsive
  • primarily handles the "look and feel" of a webpage

Javascript

Increase interactivity

  • Adds interactivity to webpage
  • Handles complex functions and features
  • Programmatic code which enhances functionality
